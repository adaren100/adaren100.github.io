**[How data science teams use ChatGPT Work](https://openai.com/academy/codex-for-work/how-data-science-teams-use-codex)** — _OpenAI · Jul 14_

**Main takeaway:** An OpenAI Academy on-demand webinar showing how data science teams use ChatGPT Work to turn questions, dashboards, and raw data into review-ready analysis assets — root-cause briefs, impact readouts, KPI memos, scoped analyses, and dashboard specs.

**Main methods:**
- **Scattered inputs → first-draft deliverables.** Starting from dashboards, metric definitions, data exports, experiment notes, and business context, ChatGPT Work assembles a first draft of the deliverable so teams can validate it before sharing.
- **Drafts arrive review-ready.** The generated draft includes charts, caveats, source links, and review questions — the framing is that the team validates the work rather than builds it from scratch.
- **Formerly the Codex app.** The webinar was recorded when these workflows lived in the former Codex app; the same flows now run in ChatGPT Work at chatgpt.com or in the ChatGPT desktop app.
- **Part of a series.** OpenAI Academy publishes parallel use-case pages and webinars for other functions (finance, sales), plus a library of ready-to-try data science use cases.

**[数据科学团队怎么用 ChatGPT Work](https://openai.com/academy/codex-for-work/how-data-science-teams-use-codex)** — _OpenAI · 7月14日_

**Main takeaway:** OpenAI Academy 的一期点播 webinar，讲数据科学团队怎么用 ChatGPT Work 把问题、dashboard 和原始数据直接变成可以送审的分析产物：根因分析简报、影响评估、KPI 备忘、限定范围的分析，还有 dashboard 规格文档。

**Main methods:**
- **零散输入直接出初稿。** 从 dashboard、指标定义、数据导出、实验笔记加上业务背景出发，ChatGPT Work 帮你把交付物的第一版拼出来，团队负责验证和把关。
- **初稿自带审阅材料。** 生成的稿子里就带着图表、caveat、来源链接和审阅问题，思路是让团队做验证，而不是从零开始写。
- **前身是 Codex app。** 这期 webinar 录制时这些 workflow 还在原来的 Codex app 里，现在同样的流程搬到了 ChatGPT Work，网页版和桌面版都能跑。
- **系列内容之一。** OpenAI Academy 给财务、销售等职能也配了对应的 use case 页面和 webinar，另外还有一套可以直接上手试的数据科学用例库。
