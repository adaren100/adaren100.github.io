**[New usage analytics and updated spend controls for enterprises](https://openai.com/index/chatgpt-enterprise-spend-controls)** — _OpenAI · Jun 18_

**Main takeaway:** OpenAI is rolling out credit usage analytics and more granular spend controls for ChatGPT Enterprise, giving admins one unified view of ChatGPT and Codex consumption across users, products, and models, plus the ability to set defaults, per-group limits, and individual overrides.

**Main methods:**
- **Unified Global Admin Console.** Brings ChatGPT and Codex credit usage into a single view with breakdowns by user, product, and model, plus trend tracking over time and identification of top users and emerging consumption patterns.
- **Cost API parity.** The same credit-usage data is exposed via a unified Cost API, so enterprises can pipe it into their own analytics or finance systems for deeper analysis.
- **Layered spend controls.** Admins set a workspace default limit, override it for specific groups, and grant individual overrides for power users — avoiding one-size-fits-all caps for the whole org.
- **In-product request flow.** End users see their consumption against their budget and can request more credits with written context, so admins approve increases on a case-by-case basis without interrupting work.
- **Builds on earlier custom-role limits.** Extends the granular credit usage limits for custom roles introduced earlier in 2026, completing the admin tooling for managing advanced-model spend.
- **Availability.** Live today for all ChatGPT Enterprise admins; end users can view their own credit usage from workspace settings.

**[ChatGPT Enterprise 上线了 credit 用量分析和更细的花费控制](https://openai.com/index/chatgpt-enterprise-spend-controls)** — _OpenAI · 6月18日_

**Main takeaway:** OpenAI 给 ChatGPT Enterprise 加了一套 credit usage analytics 和更细的花费控制。admin 现在可以在一个地方看到 ChatGPT 和 Codex 的 credit 消耗，按用户、产品、模型拆开，也能设 workspace 默认上限、按 group 改，再对个别 power user 单独放开。

**Main methods:**
- **统一的 Global Admin Console。** 把 ChatGPT 和 Codex 的 credit usage 放进同一个视图，按 user / product / model 拆开看趋势，也能直接定位重度用户和异常用法。
- **Cost API 同步开放。** 同一份 credit 数据也通过统一的 Cost API 暴露，企业可以拉进自己的财务或分析系统做更深入的拆解。
- **分层的 spend controls。** admin 给整个 workspace 设一个默认上限，针对特定 group 再覆写，再对个别 power user 做 individual override，不必为了几个人把所有人的额度都拉高。
- **产品里直接 request 额度。** 用户能看到自己用了多少、还剩多少，需要更多 credit 时可以带上说明发请求，admin 拿到 context 后再决定批不批，不会打断工作流。
- **接的是之前 custom role 的额度功能。** 在年初为 custom role 推出的 granular credit usage limits 基础上做了延伸，把企业管 advanced model 花费的工具链补齐。
- **现在就能用。** 所有 ChatGPT Enterprise admin 今天起就可以用上；普通 user 也能在 workspace 设置里查自己的 credit 用量。
