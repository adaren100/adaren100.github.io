**[Claude Science, an AI workbench for scientists, is now available](https://www.anthropic.com/news/claude-science-ai-workbench)** — _Anthropic · Jun 29_

**Main takeaway:** Anthropic launched Claude Science in beta — an AI scientific workbench for Pro, Max, Team, and Enterprise users that unifies PubMed, Jupyter, R, and HPC cluster access into a single agent environment with 60+ pre-configured biology skills, fully reproducible auditable outputs, and a reviewer agent that independently checks citations and calculations.

**Main methods:**
- **Unified research environment.** Claude Science brings together literature search (PubMed), code execution (Jupyter, R), and cluster terminal access in one place; accessible locally on macOS/Linux or via SSH/HPC login node.
- **60+ curated skills and connectors.** Pre-configured for genomics, single-cell, proteomics, structural biology, and cheminformatics; a generalist coordinating agent can spin up specialist sub-agents created by users.
- **Auditable, reproducible artifacts.** Every generated figure and manuscript includes the exact code and environment that produced it, a plain-language description, and the full message history — enabling validation and reproduction months later.
- **Rich scientific rendering.** Natively renders 3D protein structures, genome browser tracks, and chemical structures; users request figure edits in plain language (e.g., switch axis to log scale) and the agent rewrites its own code to match.
- **HPC-aware compute management.** Claude Science handles spinning up, monitoring, and pulling results from large cluster jobs (protein folding, genomics pipelines at scale), so researchers don't need to context-switch to manage compute.
- **Reviewer agent for quality checks.** A built-in reviewer agent independently checks citations and calculations, flagging and correcting errors before manuscripts are finalized.

**[Anthropic 推出 Claude Science 科研工作台](https://www.anthropic.com/news/claude-science-ai-workbench)** — _Anthropic · 6月29日_

**Main takeaway:** Anthropic 发布了 Claude Science 公测版，面向 Pro、Max、Team 和 Enterprise 用户——把 PubMed、Jupyter、R 和 HPC 集群访问整合进一个 agent 环境，内置 60+ 生物学相关技能，所有输出全程可溯源，还有专门的 reviewer agent 帮你查引用和计算错误。

**Main methods:**
- **统一科研环境。** Claude Science 把文献检索（PubMed）、代码执行（Jupyter、R）和集群终端整合进来，可以跑在本地（macOS/Linux），也能通过 SSH 或 HPC 登录节点远程用。
- **60+ 精选 skill 和 connector。** 预装了基因组学、单细胞、蛋白质组学、结构生物学、化学信息学方向的技能；顶层是个通才协调 agent，可以拉起用户自己创建的专家 sub-agent。
- **全程可溯源的产物。** 每张图、每份手稿都附上生成时用的完整代码和环境、一段白话描述以及完整对话历史——几个月后还能照着重现和验证。
- **丰富的科学可视化。** 原生渲染 3D 蛋白质结构、基因组浏览器轨迹、化学结构；用自然语言提修改意见（比如把某轴改成 log scale），agent 直接改自己的代码。
- **帮你管计算资源。** 大计算任务（蛋白质折叠、大规模基因组流水线）Claude Science 会自己递交、监控、拉回结果，研究者不用切出去专门管集群。
- **Reviewer agent 核查。** 内置 reviewer agent 独立检查引用准确性和计算结果，在手稿定稿前帮你 flag 并纠正错误。
