**[Introducing Claude Opus 4.8](https://www.anthropic.com/news/claude-opus-4-8)** — _Anthropic · May 27_

**Main takeaway:** Anthropic upgrades Claude Opus to version 4.8 — same price as 4.7, stronger across coding, agentic tasks, reasoning, and knowledge work benchmarks, with three new platform features: user-controlled effort levels, Claude Code dynamic workflows for large-scale problems, and a 3× price cut on fast mode (2.5× speed).

**Main methods:**
- **Across-the-board benchmark gains.** Opus 4.8 beats 4.7 on coding, agentic, reasoning, and professional knowledge work evals; full comparison tables and a System Card published alongside.
- **Better agentic judgment.** Early testers report Opus 4.8 is more reliable for long-running tasks — asks clarifying questions at the right moments, catches its own mistakes mid-task, pushes back on unsound plans, and builds confidence before making large-scale changes.
- **User effort control.** claude.ai users can now dial how much effort Claude puts into a response, trading speed for depth on demand.
- **Dynamic workflows in Claude Code.** New feature lets Claude Code tackle very large-scale problems by dynamically composing sub-tasks and workflows rather than single-shot code generation.
- **Fast mode price cut.** Fast mode (2.5× speed) is now 3× cheaper than for Opus 4.7, lowering the cost of latency-sensitive agentic deployments.

**[Claude Opus 4.8 发布](https://www.anthropic.com/news/claude-opus-4-8)** — _Anthropic · 5月27日_

**Main takeaway:** Anthropic 把 Claude Opus 升级到 4.8 版本——价格跟 4.7 一样，但在 coding、agentic 任务、推理和知识工作的 benchmark 上全面提升；同时推出三个平台新功能：用户可控的努力程度、Claude Code 的 dynamic workflows（处理超大规模问题）、以及快速模式 3 倍降价（2.5× 速度）。

**Main methods:**
- **全线 benchmark 提升。** Opus 4.8 在 coding、agentic、推理、专业知识工作的评测上都超过了 Opus 4.7；完整对比表和 System Card 一并发布。
- **更好的 agentic 判断力。** 早期测试者反馈 Opus 4.8 在长时间运行任务中更可靠——会在合适时机提问、在中途自己发现错误、在计划不合理时推回去、在做大范围改动前先建立信心。
- **用户可控努力程度。** claude.ai 用户现在可以调节 Claude 在一次回答上投入多少精力——按需在速度和深度之间取舍。
- **Claude Code dynamic workflows。** 新功能让 Claude Code 能处理超大规模问题，方式是动态组合子任务和工作流，而不只是单次代码生成。
- **快速模式大幅降价。** 快速模式（2.5× 速度）现在比 Opus 4.7 时代便宜 3 倍，降低了对延迟敏感的 agentic 部署的成本。
