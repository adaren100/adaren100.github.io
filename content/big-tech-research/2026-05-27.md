### Anthropic

**[Project Glasswing: An initial update](https://www.anthropic.com/research/glasswing-initial-update)** — _Anthropic · May 21_

**Main takeaway:** In one month, ~50 Project Glasswing partners using Anthropic's security-focused **Claude Mythos Preview** found **over 10,000 high- or critical-severity vulnerabilities** in systemically important software, with several partners reporting 10×+ faster bug discovery than before. Anthropic's parallel open-source scan of 1,000+ projects surfaced 6,202 high/critical bugs at a 90.6% true-positive rate after independent triage by six security firms.

**Main methods:**
- **Mythos Preview's capability profile.** First model to solve both of UK AISI's cyber ranges (multistep attack simulations) end-to-end, and top performer on ExploitBench and ExploitGym.
- **External validation.** Mozilla found and fixed 271 vulnerabilities in Firefox 150 — 10×+ more than Firefox 148 caught with Claude Opus 4.6. Cloudflare found 2,000 bugs (400 high/critical) with a false-positive rate below human testers.
- **The bottleneck has flipped.** Finding bugs is no longer the hard part; verification, disclosure, and patching are. Some open-source maintainers asked Anthropic to slow disclosures.
- **Toolkit released to qualifying customers.** Scanning skills, a harness that spins up subagents to map a codebase and triage findings, and a threat-model builder. Cisco open-sourced its Foundry Security Spec alongside.
- **Broader-access products.** **Claude Security** (Claude Enterprise public beta) used Opus 4.7 to patch 2,100+ vulnerabilities in 3 weeks. A new **Cyber Verification Program** relaxes certain safeguards for legitimate vuln research, pen-testing, and red-teaming.
- **Caveat.** Mythos-class models aren't generally released yet — no lab has misuse safeguards strong enough. The urgency is that other labs will reach this capability soon, and an unsafeguarded release would make exploiting critical software dramatically cheaper for anyone.

**[Project Glasswing 一个月战报](https://www.anthropic.com/research/glasswing-initial-update)** — _Anthropic · 5月21日_

**Main takeaway:** Project Glasswing 启动一个月，Anthropic 拉上 50 来家合作方一起用他们专门做安全研究的 **Claude Mythos Preview**，在关键基础软件里挖出了**一万多个高危和严重漏洞**，好几家合作方说找 bug 的速度直接快了 10 倍以上。Anthropic 自己另外扫了 1,000 多个开源项目，找出 6,202 个高/严重级别漏洞，经过六家独立安全公司 triage 之后真阳性率高达 90.6%。

**Main methods:**
- **Mythos Preview 的能力档位。** 第一个把英国 AISI 两个 cyber range（多步攻击模拟）完整跑通的模型，在最近出炉的 ExploitBench、ExploitGym 上也都拿了第一。
- **外部测试也对得上。** Mozilla 用它在 Firefox 150 里找到并修了 271 个漏洞，是他们之前用 Claude Opus 4.6 测 Firefox 148 时的 10 倍还多。Cloudflare 找到 2,000 个 bug（其中 400 个高/严重），false-positive rate 他们觉得比人类测试者还低。
- **瓶颈已经反转。** "找漏洞"不再是难点，反而是 verification、disclosure、patching 跟不上。已经有几个开源 maintainer 请 Anthropic 放慢披露节奏。
- **整套工具开放给 qualifying 客户。** 包括 scanning skills、能 spin up subagent 帮你 map codebase 再 triage 结果的 harness，以及一个 threat-model builder。合作方 Cisco 也顺手把自己的 Foundry Security Spec 开源了。
- **面向更广用户的产品。** **Claude Security**（Claude Enterprise 公测）3 周里已经用 Opus 4.7 修了 2,100 多个漏洞。新启动的 **Cyber Verification Program** 对合法的 vulnerability research、pen-testing、red-teaming 放开部分 safety guardrail。
- **需要注意的 caveat。** Mythos 级别的模型暂时没有公开发布，因为目前没有任何一家实验室有足够强的 misuse safeguard。他们现在就做 Glasswing，是因为别家很快也会做出同等能力的模型，到那时大规模利用关键软件漏洞的门槛会对所有人瞬间降低。
