### Aaron Levie, Box CEO

Box CEO Aaron Levie announced Box now supports previewing, editing, version management, and secure sharing of any HTML-based content — useful for working directly with agent-produced output ([link](https://x.com/levie/status/2069140445205348432)). He also commented on Sakana's newly released model, Fugu, which routes work across a mixture of models behind a single API — automatically handling model selection, delegation, verification, and synthesis. He says this mirrors how most applied AI products already build their agent harnesses, but making it accessible as a single LLM interface any developer can talk to is a notable idea, and he expects growing value in whichever layer can best route between frontier and OSS models ([link](https://x.com/levie/status/2068917230570795178)).

Box CEO Aaron Levie 宣布 Box 现在支持预览、编辑、版本管理和安全分享任何基于 HTML 的内容——特别适合直接处理 agent 产出的东西（[链接](https://x.com/levie/status/2069140445205348432)）。他还点评了 Sakana 新发布的模型 Fugu：它在单一 API 背后用一组模型做混合路由，自动完成模型选择、任务分发、验证和结果整合。他说这其实就是现在大部分应用 AI 产品搭 agent harness 的方式，但把它做成任何开发者都能直接对话的单一 LLM 接口是个亮点，他预计未来谁能在 frontier 模型和 OSS 模型之间做好路由，这一层就会越来越值钱（[链接](https://x.com/levie/status/2068917230570795178)）。

### OpenAI

OpenAI detailed its GPT-5.5-Cyber launch: the model is built for advanced, authorized defensive work — tracing vulnerable code, validating issues, developing patches, and preparing evidence for human review ([link](https://x.com/OpenAI/status/2069104286479618296)). Alongside it, OpenAI launched the Daybreak Cyber Partner Program, giving select security software and services partners Trusted Access to GPT-5.5 so their customers benefit from its defensive capabilities while direct model access stays with participating partners ([link](https://x.com/OpenAI/status/2069104290065776881)). They also detailed Patch the Planet, working with Trail of Bits, HackerOne, Calif, researchers, and maintainers to move open source security findings through to merged fixes, with human review kept central ([link](https://x.com/OpenAI/status/2069104288417390688)).

OpenAI 详细介绍了 GPT-5.5-Cyber 的发布：这个模型专门为经授权的高级防御性安全工作打造——追踪有漏洞的代码、验证问题、开发补丁，并为人工审核准备证据（[链接](https://x.com/OpenAI/status/2069104286479618296)）。同时，OpenAI 推出了 Daybreak Cyber Partner Program，让部分安全软件和服务合作伙伴获得 GPT-5.5 的 Trusted Access，使他们的客户能享受到模型的防御能力，而模型的直接访问权限仍掌握在参与的合作伙伴手中（[链接](https://x.com/OpenAI/status/2069104290065776881)）。他们还介绍了 Patch the Planet，与 Trail of Bits、HackerOne、Calif、研究人员和维护者合作，把开源安全漏洞发现一路推进到合并修复，始终把人工审核放在核心位置（[链接](https://x.com/OpenAI/status/2069104288417390688)）。

### Greg Brockman, OpenAI President

OpenAI president Greg Brockman detailed Patch the Planet, which uses frontier AI together with professional security researchers to secure critical open source projects ([link](https://x.com/gdb/status/2069182357144580584)). He also described the new Codex Security plugin for security teams — deep scans, validating findings, tracing attack paths, building threat models, generating codebase-specific patches for review, and exporting results into other tools ([link](https://x.com/gdb/status/2069128701850386834)). He noted OpenAI's models are now discovering and generating patches for critical vulnerabilities in major browsers, network infrastructure, and operating systems such as FreeBSD and the Linux kernel, and are patching projects like cURL, Go, Python, Sigstore, and pyca/cryptography ([link](https://x.com/gdb/status/2069112120206332130)).

OpenAI 总裁 Greg Brockman 详细介绍了 Patch the Planet，用 frontier AI 配合专业安全研究人员一起加固关键开源项目（[链接](https://x.com/gdb/status/2069182357144580584)）。他还介绍了面向安全团队的新 Codex Security plugin——深度扫描、验证发现、追踪攻击路径、构建威胁模型、生成针对特定代码库的补丁供审核，并能导出到其他工具中（[链接](https://x.com/gdb/status/2069128701850386834)）。他提到 OpenAI 的模型现在已经能在主流浏览器、网络基础设施以及 FreeBSD、Linux kernel 等操作系统中发现并生成关键漏洞的补丁，目前正在为 cURL、Go、Python、Sigstore、pyca/cryptography 等项目打补丁（[链接](https://x.com/gdb/status/2069112120206332130)）。
