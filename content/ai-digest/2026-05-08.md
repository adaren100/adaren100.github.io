### Anthropic

Anthropic made three notable moves on the alignment / security front. First, they're donating Petri — their open-source alignment tool — to Meridian Labs so its development can continue independently, alongside a major update that improves the adaptability, realism, and depth of Petri's tests. Second, their security bug bounty program is now public on HackerOne, opening reports and rewards beyond the previously private community. Third, they've partnered with Neuronpedia to release NLAs (neural-level something) on open models so other researchers can get hands-on. (https://x.com/AnthropicAI/status/2052494460966019137, https://x.com/AnthropicAI/status/2052466175540629965, https://x.com/AnthropicAI/status/2052435460220211397)

Anthropic 在对齐和安全方向上一次性放了三件事。第一，他们把自家 open-source 的对齐工具 Petri 捐赠给 Meridian Labs，让这个项目以后能独立演化下去，同时还推送了一个大更新，提高了 Petri 测试的适应性、真实性和深度。第二，安全赏金计划在 HackerOne 上公开了——以前只对一小圈安全研究者开放，现在任何人都可以提交漏洞并领赏金。第三，他们和 Neuronpedia 合作，把 NLAs 直接放到开源模型上，让其他研究者也能上手玩起来。(https://x.com/AnthropicAI/status/2052494460966019137, https://x.com/AnthropicAI/status/2052466175540629965, https://x.com/AnthropicAI/status/2052435460220211397)

### OpenAI

OpenAI launched a Chrome extension that significantly expands what Codex can do for coding and work. Beyond debugging browser flows, it now handles checking dashboards, running research, and updating CRMs — moving Codex closer to "the tasks that already happen in your browser." If a task needs multiple tools, Codex picks the best one for each step: plugins where they fit, Chrome when it needs a logged-in website, and combinations as needed. Under the hood it writes and runs code to navigate structured pages and complex data entry. Available today in the Codex app in all regions except EU and UK, with support coming. (https://x.com/OpenAI/status/2052480801435189708, https://x.com/OpenAI/status/2052480803318468770, https://x.com/OpenAI/status/2052480804971028879)

OpenAI 上线了一个 Chrome 插件，大幅扩展了 Codex 能干的事。除了之前的 browser flow 调试，它现在还能查看 dashboard、做调研、更新 CRM——更接近"原本就发生在浏览器里的那些日常任务"。当一项任务需要多个工具时，Codex 会按步骤挑最合适的：能用 plugin 就用 plugin，必须登录的网站就用 Chrome，必要时混着用。它的底层做法是直接写代码、跑代码，去导航结构化页面、完成复杂的数据录入流程。今天在 Codex app 里上线，覆盖除欧盟和英国外的所有区域，欧盟英国稍后支持。(https://x.com/OpenAI/status/2052480801435189708, https://x.com/OpenAI/status/2052480803318468770, https://x.com/OpenAI/status/2052480804971028879)
