### Andrej Karpathy, AI researcher

AI researcher Andrej Karpathy argued that Claude's new "join the team in Slack" style integration represents a third major redesign of LLM UI/UX, after "LLM as a website" and "LLM as a downloaded app." This new paradigm makes the model a self-contained, persistent, asynchronous entity with org-wide tools and context, working alongside teams of humans ([link](https://x.com/karpathy/status/2069547676849557725)).

AI 研究者 Andrej Karpathy 认为，Claude 这种"加入 Slack 团队"式的新交互方式，是 LLM UI/UX 的第三次重大变革，前两次分别是"LLM 作为一个网站"和"LLM 作为一个下载的 App"。这种新范式让模型变成一个自包含、持续在线、异步运作的实体，拥有全公司范围的工具和上下文，与人类团队一起协作（[链接](https://x.com/karpathy/status/2069547676849557725)）。

### Cat Wu, Claude Code & Cowork lead at Anthropic

Cat Wu, who works on Claude Code and Cowork at Anthropic, introduced "Claude Tag," calling it a paradigm shift in how Anthropic ships products — their internal version merges 65% of product PRs, and it's their first product that's natively multi-player and proactive ([link](https://x.com/_catwu/status/2069473118742331608)). She also shared a guide for configuring agent permissions for Claude Tag, and a roundup of six common customization flows that have resonated with internal and external design partners ([link](https://x.com/_catwu/status/2069484330938998993), [link](https://x.com/_catwu/status/2069486403696869555)).

Anthropic 负责 Claude Code 和 Cowork 的 Cat Wu 介绍了"Claude Tag"，称其为 Anthropic 产品研发方式的一次范式转变——他们内部版本已经能合并 65% 的产品 PR，这也是公司第一个真正支持多人协作、且能主动介入的产品（[链接](https://x.com/_catwu/status/2069473118742331608)）。她还分享了配置 Claude Tag agent 权限的指南，以及六种已经在内部用户和外部设计合作伙伴中引起共鸣的常见定制方式（[链接](https://x.com/_catwu/status/2069484330938998993)，[链接](https://x.com/_catwu/status/2069486403696869555)）。

### Thariq, Claude Code team at Anthropic

Thariq, who works on Claude Code at Anthropic, shared practical tips for managing multiple Claude-driven Slack threads: setting up a dedicated "scheduling" channel where people can tag Claude to find calendar time, having Claude react with status emojis (⏲️✅❓🛑) on top-level threads, and keeping a pinned message in personal channels that Claude updates with the status of everything ([link](https://x.com/trq212/status/2069474343512617390), [link](https://x.com/trq212/status/2069474342220820657), [link](https://x.com/trq212/status/2069474339679052144)).

Anthropic Claude Code 团队的 Thariq 分享了管理多个 Claude 驱动的 Slack 线程的实用技巧：建一个专门的"scheduling"频道，让大家可以 tag Claude 来找日程时间；让 Claude 在顶层线程上用状态表情符号（⏲️✅❓🛑）做回应；在个人频道里让 Claude 维护一条置顶消息，实时更新所有事项的状态（[链接](https://x.com/trq212/status/2069474343512617390)，[链接](https://x.com/trq212/status/2069474342220820657)，[链接](https://x.com/trq212/status/2069474339679052144)）。

### Alex Albert, Anthropic researcher

Anthropic researcher Alex Albert said the new Slack-integrated Claude workflow has completely changed how he works with the model — it feels less like using a tool and more like managing a team ([link](https://x.com/alexalbert__/status/2069470389391241314)).

Anthropic 研究员 Alex Albert 表示，这种新的 Slack 集成 Claude 工作流彻底改变了他与模型协作的方式——感觉不再是在用一个工具，而更像是在管理一个团队（[链接](https://x.com/alexalbert__/status/2069470389391241314)）。

### Claude (Anthropic)

Anthropic's Claude account detailed the rollout of Claude Tag, now in beta on Slack for Claude Enterprise and Team plans, with wider availability planned. Key features include "ambient behavior," where Claude proactively follows up on quiet threads and flags relevant context across channels and tools, and a single persistent Claude per channel that builds context over time so teammates can pick up exactly where others left off ([link](https://x.com/claudeai/status/2069468701548531895), [link](https://x.com/claudeai/status/2069468699766005847), [link](https://x.com/claudeai/status/2069468698071494976)).

Anthropic 官方 Claude 账号详细介绍了 Claude Tag 的发布：目前已在 Slack 上为 Claude Enterprise 和 Team 计划开放 beta，未来计划扩大覆盖范围。核心功能包括"ambient behavior"（主动行为）——Claude 会主动跟进陷入沉默的线程，并标记出跨频道、跨工具的相关上下文；以及每个频道由同一个持续在线的 Claude 负责，随着时间推移积累上下文，让团队成员可以无缝接续他人的工作（[链接](https://x.com/claudeai/status/2069468701548531895)，[链接](https://x.com/claudeai/status/2069468699766005847)，[链接](https://x.com/claudeai/status/2069468698071494976)）。

### Aaron Levie, Box CEO

Box CEO Aaron Levie argued that almost all AI model and agent progress is now downstream from evals — open-weight post-training, applied-layer agent improvements, and enterprise agentic deployments all come down to how well evals are built. He predicted that the ability to understand one's own workflows and measure how well agents perform in them will become a core enterprise competency ([link](https://x.com/levie/status/2069228335255949775)).

Box CEO Aaron Levie 提出，几乎所有 AI 模型和 agent 的进步现在都依赖于 evals——无论是开源模型的后训练、应用层的 agent 改进，还是企业级的 agentic 部署，最终都归结于 evals 做得好不好。他预测，理解自身工作流程、衡量 agent 在其中表现的能力，将成为未来企业的核心竞争力（[链接](https://x.com/levie/status/2069228335255949775)）。

### Swyx, AI engineer and Latent Space host

Swyx, an AI engineer and host of the Latent Space podcast, made the case that SpaceX is effectively a "neocloud + neo-lab" hybrid in a way no other company can match: SpaceX has already recouped roughly half its investment in Cursor through compute deals, with the rest covered if Composer 3 performs well. He argued this combination — simultaneously a leading model lab and a neocloud — is uniquely effective if GPU supply is well planned, regardless of how in-house training turns out ([link](https://x.com/swyx/status/2069301071965741388)).

AI 工程师、Latent Space 播客主持人 Swyx 提出，SpaceX 实际上是一种其他公司难以复制的"neocloud + 模型实验室"混合体：通过算力交易，SpaceX 已经收回了在 Cursor 上投资的大约一半，剩下一半如果 Composer 3 表现良好也能覆盖。他认为，只要 GPU 供应规划得当，同时身兼领先模型实验室和 neocloud 这两重身份，是一种极其有效的组合，无论自家训练最终结果如何（[链接](https://x.com/swyx/status/2069301071965741388)）。

Y Combinator CEO Garry Tan highlighted Linzumi as a multiplayer alternative to Codex that he called "magical for teams," noting that its founder Sean Grove previously worked on the OpenAI team that significantly reduced sycophancy in ChatGPT before leaving to start the YC-backed company ([link](https://x.com/garrytan/status/2069474420113146355)).

Y Combinator CEO Garry Tan 推荐了 Linzumi，称其为 Codex 的多人协作版替代品，"对团队来说很神奇"。他提到该公司创始人 Sean Grove 此前在 OpenAI 团队任职，曾大幅降低 ChatGPT 的阿谀倾向（sycophancy），随后离职创办了这家 YC 投资的公司（[链接](https://x.com/garrytan/status/2069474420113146355)）。
